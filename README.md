# 🏦 Bank Performance Dashboard

## 📌 Overview
A comprehensive business intelligence dashboard built for my final year project at Northumbria University. The dashboard analyses real-world banking data to surface insights on loans, transactions, and customer behaviour.

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| Power BI | Dashboard design and visualisation |
| Python | Data processing and cleaning |
| Pandas | Data manipulation |
| SQL | Data querying |

## 📊 What the Dashboard Does
- ✅ Analyses loan performance and customer accounts
- ✅ Tracks transaction patterns over time
- ✅ Visualises customer behaviour insights
- ✅ Provides actionable business intelligence for banking decisions

## 🎯 Project Highlights
- End-to-end data processing pipeline
- Interactive dashboard with real-world financial dataset
- User evaluation and usability testing conducted
- Final Year Project - BSc Business Computing, Northumbria University, 2026
